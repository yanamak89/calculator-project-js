# Calculator App --- Frontend Portfolio Project

**Live demo:** https://yanamak89.github.io/calculator-project-js/

**Repository:** https://github.com/yanamak89/calculator-project-js

## Project Overview

A simple web-based calculator application built to practice core
front‑end skills: DOM manipulation, event handling, component structure,
and JavaScript logic.
The interface is responsive and styled in a clean, modern dark theme.

## Features

-   Number and operator input via on‑screen buttons
-   Supported operations: addition (+), subtraction (-), multiplication
    (\*), division (/)
-   **Clear** button to reset input
-   Dynamic real‑time updates of the display
-   Clean, structured, maintainable JavaScript logic

## Tech Stack

-   **HTML5** --- semantic structure
-   **CSS3** --- layout, flex/grid positioning, button styling
-   **JavaScript (ES6+)** --- event listeners, calculator logic, DOM
    updates

## Project Structure

    calculator-project-js/
    │
    ├── index.html        # Main HTML file
